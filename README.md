# Find an Airbnb in the same area, every week!

I regularly use [Airbnb](https://www.airbnb.com), and stay in the same area everyÂ week. I got fed up of setting the filters over and over again, so I wrote this little bookmarklet that works out your search for you!

Change the file to set which day of the week you want to check-in on, set the number of nights to stay, your price limit, and it does the rest...

It's not pretty, but it works :)
